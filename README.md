# Task Scheduler Optimization System

## Overview

Task Scheduler Optimization System is a constraint-aware scheduling project designed to optimize task execution order while minimizing lateness and improving resource utilization.

The project combines scheduling algorithms, priority queues, greedy optimization, visualization, and KPI reporting.

---

## Features

* Task dataset creation and management
* Priority queue implementation using heap
* Task scheduling algorithm
* Deadline optimization
* KPI analysis
* CSV report generation
* Resource utilization tracking
* Multiple visualizations
* GitHub-ready project structure

---

## DSA Concepts Used

* Priority Queue (Heap)
* Greedy Algorithms
* Sorting Algorithms
* Scheduling Optimization
* Queues
* Time Complexity Analysis
